# static/internal folder

 Main Widget Javascript code provided here.

 1. demo.js: a basic demo of JSON-like format we would expect python to be able to provide suitable orderbook data in.
 2. ob.js:  Main Orderbook object class
 3. draw_ob.js: WebGPU based code for the Canvas portion of book
 4. svg_ob.js: SVG based code for the interactive overlay elements on top of the book.
